This repository contains comprehensive Jupyter Notebooks covering fundamental Python programming concepts:

### 1. Introduction to Python (`Introduction to Python.ipynb`)
This notebook covers:
- **Jupyter Notebook Fundamentals**: Understanding the interactive environment
- **Variables and Data Types**: String declarations and type annotations
- **Print Functions**: Basic output operations
- **String Operations**: Concatenation, repetition, and formatting
- **Basic Input/Output**: Working with user data
- **Operators**: Arithmetic, comparison, and logical operators
- **Control Flow**: Conditional statements (if, elif, else)
- **Loops**: For and while loops, iteration techniques
- **Functions**: Defining, calling, and working with parameters
- **Error Handling**: Basic exception management

### 2. Python Collections (`Python_Collections.ipynb`)
This notebook explores:
- **Lists**: Ordered, changeable collections with duplicates
  - Creating and accessing lists
  - List indices and slicing
  - Common list methods (append, remove, pop, etc.)
  - List comprehensions
- **Tuples**: Ordered, immutable collections
  - Tuple operations and unpacking
  - Use cases for immutability
- **Sets**: Unordered collections without duplicates
  - Set operations (union, intersection, difference)
  - Mathematical set operations
- **Dictionaries**: Key-value pair collections
  - Accessing and modifying dictionary data
  - Dictionary methods
  - Practical applications

## üöÄ Getting Started

### Prerequisites
- Python 3.x installed on your system
- Jupyter Notebook or JupyterLab
- Basic computer programming knowledge

### Installation

1. **Install Python** (if not already installed):
   - Download from [python.org](https://www.python.org/downloads/)
   - Verify installation: `python --version`

2. **Install Jupyter Notebook**:
   ```bash
   pip install jupyter
   ```

3. **Clone or download this repository** to your local machine

4. **Launch Jupyter Notebook**:
   ```bash
   jupyter notebook
   ```

5. **Open the notebooks** in your browser and start learning!

## üîß Troubleshooting

### Common Issues:

**Jupyter won't start:**
- Ensure Python and Jupyter are correctly installed
- Try: `pip install --upgrade jupyter`

**Kernel issues:**
- Restart the kernel: `Kernel ‚Üí Restart`
- Clear output and restart: `Kernel ‚Üí Restart & Clear Output`

**Import errors:**
- Install missing packages: `pip install package_name`

**Code doesn't run:**
- Check for syntax errors (look for red highlights)
- Ensure previous cells are executed in order
